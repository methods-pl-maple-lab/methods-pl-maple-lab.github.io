How to add yourself to People

Adding yourself to the people page is a simple matter of:

  1. Forking this repository,
  2. Adding the appropriate folders and files to your fork,
  3. Editing the files to reflect the information you want in them, and
  4. Sending a pull request to the website to build and merge.

You can edit your fork directly on GitHub, or you can clone the forked repository to your local machine. The instructions below include information for either approach.

  1. Create a folder under people using your surname (if you have more than one, use underscores for spaces).
    For example, if we assume you are working from the top level of this repository on a Mac and your name is Xena Warrior Princess, running
    mkdir src/content/people/warrior_princess
    should do the trick.
    You can also manually
  2. Add a headshot named headshot.jpg to the folder you just created. The file must have the name headshot.jpg or it will not render.
  3. Copy this file into your new people directory as index.md, e.g., cp src/content/people/instructions.md src/content/people/warror_princess/index.md.
  4. Edit the front matter of this file (i.e., the stuff between the two +++ strings) as appropriate. This file includes all of the possible information you can include. If you do not want to include a piece of information, you should delete that entry. For example, you may not have any publicly available publications yet.
